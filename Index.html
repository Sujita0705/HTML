What is HTML ?

HTML, or HyperText Markup Language, is the standard language used to create web pages. It's the basic building block of the web.HTML is used to structure content, such as text,links, and images,on a web page.

How does HTML work?

HTML is made up of elements, such as tags and attributes, that tell the browser how to display content HTML is used alongside CSS and JavaScript to create websites HTML files are text-based files.
